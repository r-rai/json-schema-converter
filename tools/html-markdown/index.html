About This Tool

Convert between HTML and Markdown formats instantly. Supports tables, code blocks, and more.

Privacy First Client-Side Only Open Source

help How to Use This Tool

Quick Start

  1. HTML to Markdown: Paste HTML → Click "Convert to Markdown" → Markdown appears
  2. Markdown to HTML: Paste Markdown → Click "Convert to HTML" → HTML appears
  3. Copy Result: Use Copy button to copy output to clipboard

Supported Features

  • Headings, paragraphs, lists (ordered & unordered)
  • Links, images, code blocks, inline code
  • Tables, blockquotes, horizontal rules
  • Bold, italic, strikethrough text

Tips

  • Use options panel to customize conversion behavior
  • Preview mode shows rendered HTML for Markdown input
  • All HTML is sanitized to prevent XSS attacks

lock All processing happens in your browser. No data is sent to any server. Your privacy is guaranteed.
