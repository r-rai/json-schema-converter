About This Tool

Compare two text files and see differences highlighted. Perfect for code reviews and document comparison.

help How to Use This Tool

Quick Start

  1. Add Original Text: Paste original content in left box
  2. Add Modified Text: Paste modified content in right box
  3. Compare: Click "Compare Texts" to see differences
  4. Review Results: Green = additions, Red = deletions, Gray = unchanged

Understanding Colors

  • Green: Lines added in modified version
  • Red: Lines removed from original version
  • Gray: Lines unchanged between versions

Tips

  • Perfect for code reviews and document comparison
  • Statistics show total lines added/removed
  • Clear buttons reset both sides for new comparison
